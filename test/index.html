Input Radio Or Check To Image

Here is a demo of the jquery plugin available on github

Input with checkbox 

                
$(function()
{
    $('[name="check_1"]').ircti();
});

Input with radio 

                
$(function()
{
    $('[name="radio_1"]').ircti();
});

Input change options 

                
$('[name="radio_2"]').ircti({
    height: "20px",
    width: "20px",
    url_img_check: "../img/checkbox_check_2.png",
    url_img_no_check: "../img/checkbox_no_check_2.png"
});
